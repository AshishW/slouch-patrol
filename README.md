> **"Because your spine isn't a laptop stand."**


Slouch-Patrol is a Chrome Extension built to remind you to sit straight, stretch and hydrate when you are actually at your desk.

## Key Features

* **Animated In-Page "Peeks"**: An animated patrol officer physically appears on your current webpage to deliver a randomized wellness message.
* **Customizable Intervals**: User-friendly options page to persist your settings across all your devices.
* 

## Installation for Testing

1. Clone this repository.
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable **Developer Mode** (top right).
4. Click **Load unpacked** and select the project folder.
5. Pin the extension and right-click to access **Options** to set your patrol frequency.
